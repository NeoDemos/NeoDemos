LIMITATIONS & WORKAROUNDS
================================================================================

LIMITATION 1: Voting Records Not Available
  Impact: Can't show how councillors voted
  Workaround: Display decision outcomes from official documents
  User Communication: "Decision outcomes shown; detailed voting records 
                       not available in public API"

LIMITATION 2: Minutes (Notulen) Incomplete
  Impact: Some meetings lack formal minutes
  Workaround: Use agenda items + decision documents as proxy
  Enhancement Option: Web scrape rotterdam.nl for published notulen PDFs
                      (requires separate implementation)

LIMITATION 3: Index Name Is Date-Stamped
  Problem: Index name changes periodically → code breaks
  Solution: Implement dynamic index discovery (see ORI_INTEGRATION_GUIDE.md)
  Effort: ~2 hours of development

LIMITATION 4: PDF Text Extraction Quality
  Issue: OCR errors in older PDFs, lost formatting
  Acceptable For: Summarization, keyword search, AI analysis
  Not Suitable For: Direct quotes, precise layout analysis

Required Fix:
  Replace hardcoded index with dynamic discovery:
  
  async def get_rotterdam_index():
      async with httpx.AsyncClient() as client:
          response = await client.get(
              "https://api.openraadsinformatie.nl/v1/elastic/_cat/indices"
          )
          indices = response.text.split('\n')
          rotterdam_indices = [
              i for i in indices 
              if 'rotterdam' in i and 'ori_' in i
          ]
          return sorted(rotterdam_indices)[-1].split()[0]

Estimated Time: 1-2 hours (including testing)

================================================================================
API ENDPOINT REFERENCE
================================================================================

Base URL: https://api.openraadsinformatie.nl/v1/elastic

Query Structure:
  Method: POST
  Content-Type: application/json
  
  {
    "query": {
      "bool": {
        "must": [
          { "term": { "_index": "ori_rotterdam_20250629013104" } },
          { "term": { "@type": "Meeting|AgendaItem|Document" } },
          { "range": { "start_date": { "gte": "2024-01-01T00:00:00Z" } } }
        ]
      }
    },
    "size": 50,
    "sort": [{"start_date": "asc"}]
  }

Document Types:
  - Meeting: Council/committee sessions (1,931 total, 584 in 2024+)
  - AgendaItem: Individual agenda points (linked to meetings)
  - Document: PDFs/attachments with pre-extracted text

Key Fields:
  Meeting: id, name, start_date, committee, status, location, agenda[]
  AgendaItem: id, name, position, parent, attachment[], decision_type
  Document: id, name, original_url, file_name, text[], content_type

Rate Limits: Not documented, recommend max 100 results per query

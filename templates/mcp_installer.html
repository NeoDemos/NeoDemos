NeoDemos MCP Server

Verbind Claude, ChatGPT of andere AI-assistenten met 100.000+ Rotterdamse raadsdocumenten via het Model Context Protocol.

Geen token nodig! De MCP-server gebruikt OAuth 2.1 — Claude opent automatisch een inlogvenster bij de eerste verbinding. U hoeft alleen de server-URL toe te voegen.

1 Open Claude Desktop

Ga naar Settings → Connectors (of Developer → Edit Config in oudere versies).

2 Voeg de server toe

Klik op Add Integration en plak de volgende URL:

{{ mcp_url }}

Of voeg het handmatig toe aan uw claude_desktop_config.json:

{
  "mcpServers": {
    "neodemos": {
      "url": "{{ mcp_url }}"
    }
  }
}

3 Inloggen

Bij de eerste keer dat u een NeoDemos-tool gebruikt, opent Claude een browservenster om in te loggen op neodemos.nl. Na toestemming werkt alles automatisch.

{% if not user %}
Nog geen account? Registreer hier zodat u klaar bent wanneer Claude om inlog vraagt.
{% endif %}

1 Voer dit commando uit in uw terminal

claude mcp add neodemos --url {{ mcp_url }}

Claude Code handelt de OAuth-flow automatisch af bij eerste gebruik.

1 Open ChatGPT Desktop

Ga naar Settings → Beta Features → MCP Servers.

2 Voeg de server toe

Klik op Add Server en voer de URL in:

{{ mcp_url }}
Le Chat (Mistral) is de enige grote AI-assistent met MCP-ondersteuning in het gratis abonnement. Beschikbaar op desktop, iOS en Android. Volledig EU-gehost.

1 Open Le Chat

Ga naar chat.mistral.ai en log in.

2 Navigeer naar Connectors

Klik op het Intelligence-icoon (bol) in de linkerzijbalk → Connectors → + Add Connector → Custom MCP Connector.

3 Voer de URL in

{% if user %}

Gebruik de authenticatie-URL (volledige toegang, 20 tools):

{{ mcp_url }}
{% else %}

Zonder account — gebruik de publieke URL (gratis, geen login, 20 tools, snelheidslimiet):

{{ public_mcp_url }}

Of maak een account aan voor de authenticatie-URL met hogere limieten.

{% endif %}

4 Autoriseer de verbinding

{% if user %}

Klik op Connect. Le Chat opent een browservenster voor de OAuth-inlog op neodemos.nl. Na toestemming verschijnen de NeoDemos-tools in Le Chat.

{% else %}

Klik op Connect. De publieke URL heeft geen login nodig — de tools zijn direct beschikbaar.

{% endif %}

Problemen? Zie de installatiegids voor troubleshooting.

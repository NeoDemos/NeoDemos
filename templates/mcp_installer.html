NeoDemos MCP Server

Verbind Claude of ChatGPT met de Rotterdamse raadsinformatie via het Model Context Protocol (MCP).

Claude Desktop

Gebruik NeoDemos direct in Claude Desktop met automatische configuratie.

ChatGPT

Verbind NeoDemos met ChatGPT via de MCP-integratie.

Handmatige Configuratie

Claude Desktop

  1. Open Claude Desktop en ga naar Settings → Developer → Edit Config.
  2. Voeg het volgende toe aan uw claude_desktop_config.json:
{
  "mcpServers": {
    "neodemos": {
      "url": "https://mcp.neodemos.nl/mcp",
      "headers": {
        "Authorization": "Bearer UW_API_TOKEN"
      }
    }
  }
}

Claude Code (CLI)

  1. Voeg de MCP-server toe via de CLI:
claude mcp add neodemos \
  --transport streamable-http \
  --url https://mcp.neodemos.nl/mcp \
  --header "Authorization: Bearer UW_API_TOKEN"

ChatGPT Desktop

  1. Open ChatGPT Desktop en ga naar Settings → Beta Features → MCP Servers.
  2. Klik op Add Server en voer in:
Server URL: https://mcp.neodemos.nl/mcp
Authorization: Bearer UW_API_TOKEN
{% if user %} Vervang UW_API_TOKEN met uw persoonlijke API-token. U kunt een token aanmaken op de Instellingen pagina. {% else %} U moet eerst inloggen om een API-token aan te maken. Na het inloggen kunt u op de Instellingen pagina een token genereren. {% endif %}
